## python
- 使用`for`循环
```
for i in range(3)
    print("hello world)
```

`range`表示循环的次数

- `if`判断语句格式如下

```python
a=10
b=10
if a>b:
    print("a>b")
elif a<b:
    print("a<b")
else:
    print("a=b")
```

注意，在python中，else if可以直接写成`elif`
- while循环
  语法如下
  
  ```
  a=5
  while a<4:
      print(a)
      a-=1
  ```
